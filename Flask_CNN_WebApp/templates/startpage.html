Real-Time Sign Detection and Recognition

In this Project, initially we are going to trained a model using sign images. Then by using the model we are going to detect the real-time sign actions performed by the user. This work main focus is on creating a vision-based system able to do real-time sign language recognition. The reason for choosing a system based on vision relates to the fact that it provides a simpler and more intuitive way of communication between a human and a computer.
